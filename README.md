# index-webs-lab

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放、归档以及（可选）发布各种独立开发的 HTML 页面。这些页面可能涉及不同的主题或功能，彼此之间没有强关联。仓库本身不针对任何特定网站或域名，仅作为页面文件的集中管理空间。

## 目录说明

```
index-webs-lab/
├── pages/          # 存放独立 HTML 页面的主目录
│   ├── example/    # 示例页面文件夹（可删除或替换）
│   └── ...
├── assets/         # 公共资源文件（样式、脚本、图片等）
├── docs/           # 文档或说明文件（可选）
└── README.md       # 本文件
```

- `pages/`：每个子文件夹代表一个独立页面项目，包含其 HTML、CSS、JavaScript 及相关资源。
- `assets/`：供多个页面共享的静态资源，避免重复存储。
- `docs/`：可选的额外说明文档。

## 页面归档说明

- 每个页面应放置于 `pages/` 下独立文件夹中，命名尽量简洁明了。
- 页面内引用的资源尽量使用相对路径，确保文件夹可独立移动或部署。
- 本仓库不保证所有页面长期可用，部分页面可能仅为实验性或临时用途。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来改进已有页面或添加新页面。
- 提交新页面时，请在 `pages/` 下新建文件夹，并确保页面能独立运行。
- 如果页面需要特定环境（如后端服务），请在页面文件夹内添加 `README.md` 说明。
- 仓库维护者会不定期清理不再维护或过时的页面。
